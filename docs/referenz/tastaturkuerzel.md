# Tastaturkuerzel

ComplianceOS unterstuetzt Tastaturkuerzel fuer schnelle Navigation.

## Navigation

| Kuerzel | Aktion |
|---------|--------|
| ++ctrl+k++ | Command Palette oeffnen |
| ++ctrl+1++ | Dashboard |
| ++ctrl+2++ | Chat |
| ++ctrl+3++ | Audits |
| ++ctrl+4++ | Findings |
| ++ctrl+5++ | Dokumente |
| ++ctrl+6++ | Policies |
| ++ctrl+7++ | Reports |
| ++ctrl+8++ | Einstellungen |

## Command Palette

Die Command Palette (++ctrl+k++) ermoeglicht schnellen Zugriff auf alle Funktionen:

- Tippen Sie den Namen einer Seite oder Funktion
- Waehlen Sie mit Pfeiltasten und ++enter++
- ++escape++ zum Schliessen

## Chat

| Kuerzel | Aktion |
|---------|--------|
| ++enter++ | Nachricht senden |
| ++shift+enter++ | Zeilenumbruch |
